How It Works
Each identity namespace (GitHub, Twitter, ERC-8004) runs a dedicated provider that returns trust signals with a score and confidence level. TrstLyr fuses these using Subjective Logic Cumulative Belief Fusion — each signal is an opinion tuple (belief, disbelief, uncertainty). High-confidence signals dominate; vacuous signals (no data) don't corrupt the result.
